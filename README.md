# hexo 


## install hexo

```bash
npm install -g hexo-cli
npm install hexo --save
```

## theme

``` bash
git clone git@github.com:hangyan/maupassant-hexo.git themes/maupassant
npm install hexo-renderer-pug --save
npm install hexo-renderer-sass --save
```
